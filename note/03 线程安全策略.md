# 线程安全策略

## 不可变对象
不可变对象需满足的条件
+ 独享创建以后其状态就不能修改
+ 对象所有域都是final类型
+ 对象是正确创建的（在对象创建期间，this引用没有溢出）

**final** 关键字：类、方法、变量
+ 修饰类：不能被继承
+ 修饰方法：1、锁定方法不能被继承类修改；2、效率 
+ 修饰变量：基本数据类型变量、引用数据类型变量

**Collections.unmodifiableXXX:Collection、List、Set、Map.....**


**Guava:ImmutableXXX:Collection、List、Set、Map......**

## 线程封闭
+ 堆栈封闭：局部变量，无并发问题
+ ThreadLocal 线程封闭：特别好的线程封闭方法
